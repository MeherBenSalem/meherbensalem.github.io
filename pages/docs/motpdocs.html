Introduction

  • Project Name: Memories of the Past - RPG leveling system
  • Version: v0.11.0
  • Author: NAIZO
  • License: ARR

Welcome to the Memories of the past Mod, a unique and engaging addition to Minecraft that introduces a dynamic leveling system through the collection and use of Memory Fragments. Experience a new dimension of gameplay as you unlock abilities, access special content, and face new challenges!

Important Note

Ensure that your configuration is properly set up to avoid loot-related bugs.

Requirements

Required software to run and configure the Mob Loot Bags mod:

  1. Minecraft 1.18.2, 1.19.2, 1.20.1, 1.21.1
  2. Forge or NeoForge
  3. A text editor for modifying the config files

Leveling Configuration

                                    ["Leveling Config"]
                                    lesser_experience_value = 20.0
                                    better_experience_value = 200.0
                                    greater_experience_value = 2000.0
                                    level_per_orb = 1.0
                                    scale_factor = 30.0
                                    points_per_level = 1.0
                                    enableDrops = true
                                    onDeathReset = false
                                    maxPlayerLevel = 500.0

These settings define how experience is awarded and how levels are scaled in your mod.

  • lesser_experience_value: The value of xp received from the lesser fragments of knowledge
  • better_experience_value: The value of xp received from the better fragments of knowledge
  • greater_experience_value: The value of xp received from the lesser fragments of knowledge
  • level_per_orb : This setting defines how many levels a player gains per orb of leveling .
  • scale_factor: The scale factor is used to adjust the experience required for level progression.
  • points_per_level: This setting indicates the number of points players earn for each level gained
  • enableDrops: This option determines whether or not loot drops are enabled upon leveling or not. Setting this to true allows players to receive drops of knowledge from mods when they die
    • else it will automatically claim the xp.
  • onDeathReset: This setting indicates if the stats reset when the player dies
  • maxPlayerLevel: Max level the player can reach
