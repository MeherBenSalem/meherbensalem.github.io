Configurations

Welcome to our guide on editing the configuration files for the Ascendant Mobs mod. This tutorial will help you understand how to adjust settings that control mob difficulty as you explore further into the Minecraft world.

Distance Scale Settings

main_config.toml file

The scale_distance setting controls how mob levels increase as you move away from the spawn point:

                    [Distance Scale Settings]
                    #Recommended over 25 000
                    scale_distance = 55000.0
                    #The scale type can be 'vertical', 'horizontal', 'both', 'random' or 'time'
                    scale_type = "time"
                    day_factor = 1.0
                    random_min = 0.0
                    random_max = 200.0

In this configuration:

  • scale_distance is set to 55,000 blocks, meaning mobs will gradually become stronger as you move further from the spawn point. Adjust this value to balance the game’s challenge level.
  • scale_type determines the method of scaling. You can choose from:
    • - vertical: Scales mob levels based on vertical distance from world spawn point (Y-axis).
    • - horizontal: Scales mob levels based on horizontal distance from world spawn point (X and Z axes).
    • - both: Scales mob levels based on distance from world spawn point in all directions (X, Y, and Z axes).
    • - random: Applies a random scaling factor within a specified range.
    • - time: Adjusts mob levels based on the in-game time or player's exploration duration.
  • day_factor, random_min, and random_max further fine-tune the scaling effect.

Mob Levels Settings

Control mob base levels and scaling factors here:

                    [Mobs Levels Settings]
                    base_level = 0.0
                    scale_factor = 0.5
                    base_level_nether = 50.0
                    base_level_end = 100.0
                    random_effects = true
                    random_effects_level = 100.0
                    random_effects_chance = 10.0
                    base_level_territory = 75.0
                    max_level_overworld = 75.0
                    max_level_nether = 100.0
                    max_level_end = 200.0

In this configuration:

  • base_level sets the default level of mobs near the spawn.
  • scale_factor controls the increase in mob levels with distance.
  • base_level_nether and base_level_end define base levels for the Nether and End dimensions, respectively.
  • random_effects enables random buffs for mobs at certain levels. Customize the effects' level and chance with random_effects_level and random_effects_chance.
  • max_level_overworld, max_level_nether, and max_level_end set maximum levels for mobs in each dimension.

Blacklist Settings

To exclude specific mobs from the leveling system, use the list_config.toml file: 

                    [Blacklist]
                    #write the registry name of the entities
                    banned = ["minecraft:zombie"]
                    death_messages = []

In this configuration:

  • banned lists entities to exclude from leveling. In this example, zombies are banned.
  • death_messages allows exclusion of messages shown when banned mobs are defeated.
